Ridmik Board is basically a mod of Ridmik Bangla Keyboard. Ridmik Keyboard is a Bengali and English Keyboard for Android 2.3+. Original app wasn't being updated for long time. So I thought about bringing few changes, and make it more user friendly. That's all. Enjoy!
